## Purpose of this website
This introductory webpage serves as a personal portfolio. Its purpose is to demonstrate basic HTML knowledge,
including HTML structure and CSS skills that I have learned as part of my Computers Applications course CMPA-3301 at Texas Tech University.
It comprises the following files: index.html, style.css, README.md and profile.jpg and texastechlogo.png.
The webpage shows information intended to introduce myself, my name, my picture, my major and minor, as well as my professional interests. It explains my goals in this course as well as my goals for my future professional career. It also includes my hobbies and my GitHub link.
By creating and publishing this webpage, I am also demonstrating I have learned how to use Visual Studio Code, GitHub Desktop and GitHub.com. One of the goals of Project 02 is for students to be able to create a webpage and be able to publish it, so it's live and accessible to the public. Another learning objective is to use the GitHub version control system, where every time there is a file created, uploaded or updated, the user types and records a commit, keeping track of every change that has been made and the date when it was made. So it keeps me accountable.
